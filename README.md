# README

Simple CRUD ROR Blogging Application

## Running the Application
1. Open terminal.
1. Navigate to "SimpleBlog" project folder.
1. Run: `$ rails s` or `$ rails server`
1. Open a web browser and go to [https://localhost:3000](https://localhost:3000).
